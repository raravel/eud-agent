## Description
Replace the panel WebSocket client with a Tauri IPC client (`panel/src/lib/ipc.ts`) mapping the panel's
v2 chat schema (decision 13 / feature 15) to `invoke` + `listen`: commands
chat/plan_feedback/plan_approve/changeset_decision/cancel/reset/status/list -> `invoke`; events
agent_event/answer/plan/changeset/rollback_result/progress/error -> `listen` (status/list resolve as
command return values). Wire the store; drive its lifecycle hooks from IPC readiness (no reconnect loop —
the single in-process app has no socket). Move the shared protocol TYPES (ClientMessage/ServerMessage
unions + guards, and the shared shapes FileEntry/ChangesetItem/Diagnostic/ProgressStage) out of
`ws/protocol.ts` into `lib/ipc.ts` (or a sibling types module) so the store and components keep their
imports; update `App.tsx` to use the IPC client; delete `panel/src/ws/*`. Remove all
WS/token/Origin/server.ready code.
