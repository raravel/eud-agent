## Context

The `search_docs` tool existed as a validation-only stub (TODO EUD-055+: returns `[]`)
while the engine's `[reference context]` section already ran the in-process
`rag.search`. Separately, codex was observed writing ENGLISH `search_docs`
queries against the ECA corpus, which is Korean community material — bge-m3 is
cross-lingual, but community jargon (음수 로케이션, 피탄판정, 비비기 ...) has
no English form, so English queries lose recall.

     `(query, k) -> list[dict]`; app.py wires `rag.search` with `cfg.rag_db`
