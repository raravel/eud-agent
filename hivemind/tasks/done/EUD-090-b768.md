## Description
Refactor eud-agent from the POC topology (external Python FastAPI server spawned by the
Lua bridge + WebView2 hosted inside the editor) into a single standalone Tauri 2 desktop
app with an in-process Rust backend, a statically linked C++ map engine, and an in-process
fastembed RAG. This is the v2 architecture re-plan.
