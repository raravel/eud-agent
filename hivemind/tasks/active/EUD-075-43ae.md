## Description
Epic: give codex a per-map-project memory harness it updates autonomously while working —
resource ledger, file-role map, conventions, lessons — injected into every prompt as a
`[project memory]` section, reviewed through the existing changeset flow, and editable in a
panel memory view. Eliminates the per-chat amnesia that makes codex re-derive (or collide
with) the map's resource allocations.
