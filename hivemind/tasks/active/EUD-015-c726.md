## Description
bridge_io module: send() writes inbox srv-uuid8.cmd UTF-8 without BOM, polls outbox srv-uuid8.result every 0.2s, deletes the result after reading. Timeout 10s, extended to 180s while status.txt says compiling=true, with an on_busy callback. On timeout leave the .cmd in place and raise BridgeBusy. Helpers: ping/status/list_files/get/set/neweps; list_files parses path TAB ftype into {path, ftype, settable}. cleanup_stale() removes leftover srv-* files at startup. Never touch agent_* files (legacy runner namespace).
