## Description
S8 groups the Rust map-tool integration layer that consumes the completed low-level isom FFI
(S2: EUD-101..106) and the chk parser (EUD-105). Today those are dead/stubbed: `isom` is not a
`src-tauri` dependency, mapsafe has only a FakeEngine, `chk::digest_chk` has no caller, and the
map tools have schemas but no handlers. This story wires the real engine and the three map tools.
