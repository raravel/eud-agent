## Description
Server lifecycle in the bridge. Init: read agent.cfg (File.ReadAllText + simple string extraction of the 3 flat keys), delete pre-existing server.ready, spawn python -m eud_agent via luanet ProcessStartInfo (UseShellExecute=false, CreateNoWindow=true, WorkingDirectory repo_root\server), keep the Process object in a global. Tick: write heartbeat.txt FIRST, unconditionally, before the IsCompilng early-return; validate server.ready (pid equals owned proc Id - never Process.GetProcessById; mtime after bridge start); respawn on proc.HasExited with a 30s throttle. Fix the stale "v5 loaded" marker to v7. Degrade to v6 file-IPC-only behavior when agent.cfg is missing.
