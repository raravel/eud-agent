## Description
Replace the v6 WPF control panel with a WebView2-hosted panel window. Load Core+Wpf assemblies (DLLs deployed next to the editor exe), create Window + WebView2 control, set CreationProperties.UserDataFolder to Data\agent\webview2 (never the default next-to-exe folder), EnsureCoreWebView2Async, Navigate to http://127.0.0.1:port/?token=... using the validated server.ready values. Subscribe NavigationCompleted: IsSuccess false means re-Navigate on a later Tick with 3s backoff (WebView2 never auto-retries). Re-arm: track the window handle; while project open AND window not alive, recreate next Tick (editor closes aux windows on project switch). PANEL command now shows/refocuses this window; remove the WPF showPanel body. Korean labels via u8().
