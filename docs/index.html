EUD Editor 3 외부 AI 에이전트

자연어로 epScript를 작성하고
에디터에 바로 적용하세요

eud-agent는 StarCraft EUD 맵 제작을 위한 데스크톱 AI 에이전트입니다. 지시를 epScript(eps) 코드로 변환하고, RAG 근거와 함께 EUD Editor 3에 안전하게 적용합니다.

Windows용 다운로드 … GitHub에서 보기

Windows 10/11 (64-bit) · 무료 · 자동 업데이트

시작하기 전에

설치 전 아래 항목을 확인하세요.

Windows 10/11

64-bit 데스크톱 환경에서 동작합니다.

WebView2 런타임

보통 Windows에 기본 포함됩니다. 없을 경우 여기서 설치하세요.

EUD Editor 3

별도로 보유한 EUD Editor 3가 필요합니다. 설치 후 에디터를 실행하면 자동으로 연결됩니다.

codex CLI

코드 생성에 codex CLI를 사용합니다. 직접 설치·로그인(BYO)이 필요합니다.

최초 실행 시 임베딩 모델(bge-m3)과 RAG 인덱스를 자동으로 내려받습니다(1회, 약 600MB 이상). 다운로드가 끝나면 이후 실행은 즉시 시작됩니다.

설치 방법

네 단계면 끝납니다.

  1. 1

    설치 파일 다운로드

    위의 Windows용 다운로드 버튼으로 최신 -setup.exe 설치 프로그램을 받습니다.

  2. 2

    설치 프로그램 실행

    내려받은 파일을 실행합니다. 사용자 단위(현재 사용자) 설치라 관리자 권한이 필요 없습니다.

    코드 서명이 적용되지 않아 Windows SmartScreen 경고가 나타날 수 있습니다. 추가 정보 → 실행을 눌러 진행하세요.

  3. 3

    최초 실행 설정

    앱을 처음 켜면 설정 화면에서 임베딩 모델과 RAG 인덱스를 자동으로 내려받습니다. 완료될 때까지 잠시 기다리세요.

  4. 4

    에디터 연결

    EUD Editor 3를 실행하면 브리지가 자동으로 설치·연결됩니다. 이제 패널에서 지시를 입력하고 코드를 적용할 수 있습니다.

자동 업데이트

설치 후에는 앱이 서명된 업데이트를 자동으로 확인하고 적용합니다. 내려받은 모델과 설정은 업데이트 시에도 그대로 보존됩니다.
